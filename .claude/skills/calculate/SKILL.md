# /calculate -- Расчёт стоимости отправления

## Алгоритм

1. Получи параметры: индекс отправителя, индекс получателя, вес в граммах
2. Вызови MCP tool `calculate` с параметрами
3. Покажи стоимость, НДС и сроки доставки

## Формат ответа

- Стоимость (в рублях, пересчитать из копеек)
- НДС
- Сроки доставки (мин-макс дней)

## Примеры

    /calculate 101000 630099 500
    /calculate 190000 420000 1000 EMS
