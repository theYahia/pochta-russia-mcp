# /track -- Отслеживание посылки Почты России

## Алгоритм

1. Получи трек-номер от пользователя (формат: RA123456789RU, EE123456789RU и т.д.)
2. Вызови MCP tool `track` с параметром barcode
3. Покажи последний статус и историю перемещений

## Формат ответа

- Последний статус (операция, дата, место)
- Полная история событий (таблица)

## Примеры

    /track RA123456789RU
    /track EE987654321RU
